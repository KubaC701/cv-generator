# Open Source Dashboard

**Type:** Personal project
**Status:** Active

## Overview

A real-time analytics dashboard for monitoring open source project health. Aggregates data from GitHub, npm, and social mentions into a single view. Built to scratch a personal itch — maintaining 3 OSS libraries and wanting a unified view of activity.

## Key Features

- Real-time GitHub stats (stars, forks, issues, PR velocity)
- npm download trends and version adoption tracking
- Social mention aggregation from Twitter/Reddit

## Skills & Technologies

- **Languages:** TypeScript
- **Frameworks:** Next.js, D3.js
- **APIs:** GitHub API, npm registry, Reddit API
- **Infrastructure:** Vercel, Upstash Redis

## Motivation

Built to solve a personal workflow problem. Demonstrates full-stack development, API integration, and data visualization skills.
